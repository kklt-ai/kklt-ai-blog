# 首页作品直达导航设计

## 目标

让用户无需滚动到首页作品区，即可从顶部导航直接进入“md生成卡片”和“自媒体封面”。桌面端和手机端都提供入口。

## 设计

- 桌面端保留现有“作品”下拉菜单，将下拉内容替换为两个项目链接：
  - “md生成卡片”指向 `/md-card`
  - “自媒体封面”指向 `/cover`
- 手机端在顶部增加紧凑的“作品”下拉按钮，提供相同的两个项目链接。
- 保留语言切换和“关于”链接的现有行为。
- 下方作品卡片及其链接保持不变，继续承担项目介绍作用。
- 中英文界面中的项目名称沿用 `homeCopy` 里的现有卡片标题，避免重复维护文案。

## 组件边界

改动限定在 `src/home/components/HeaderNav.tsx`。导航组件从现有 `copy.cards` 生成项目入口，不增加新的状态、配置或组件抽象。

## 交互与响应式行为

- 桌面端沿用现有悬停及键盘聚焦时展开下拉菜单的交互。
- 手机端使用可点击、可聚焦的下拉入口，并保持顶部布局不溢出。
- 两个项目均为站内普通链接，在当前页面打开。

## 验证

更新 `src/app/page.test.tsx`，验证桌面端和手机端导航均包含两个入口，并且链接分别指向 `/md-card` 与 `/cover`。运行该测试文件确认行为。
